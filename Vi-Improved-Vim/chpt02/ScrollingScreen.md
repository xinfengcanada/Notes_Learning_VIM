# Vim 翻屏操作

在这篇文章里，我将和大家分享有关Vim翻屏的各种操作。其它的不多说，先来看看命令：

- ctrl-u 上翻半屏
- ctrl-d 下翻半屏
- ctrl-f 下翻一屏
- ctrl-b 上翻一屏
